# Manual Testing of Login Functionality

## Project Description
This project demonstrates manual testing skills applied to a login functionality.
The goal is to design test cases, execute them, and document discovered defects.

## Scope
- Functional testing of login form
- Positive and negative test scenarios
- Input validation testing

## Test Artifacts
- Test cases
- Bug reports
